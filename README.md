## Featured Projects

### Infant Pose Refinement and Movement Analysis

A deep learning framework for infant pose estimation, pose refinement, temporal movement analysis, and explainable AI using YOLO11 and MediaPipe.

**Technologies:** Python, PyTorch, YOLO11, MediaPipe, Computer Vision, Temporal Deep Learning, Explainable AI

---

### Explainable Traffic Congestion Prediction System

An intelligent traffic forecasting system that combines historical traffic data, live traffic conditions, weather information, graph-based deep learning models, explainable AI, counterfactual analysis, and a user-friendly chatbot.

**Technologies:** Python, DCRNN, Graph WaveNet, Graph LSTM, Transformers, SHAP, Weather APIs, Traffic APIs, LLM

---

---

### Emotion Recognition Using BERT

A multi-label emotion recognition system built using the GoEmotions dataset, DistilBERT, BiLSTM with attention, TextCNN with attention, and SHAP-based explainability.

**Technologies:** Python, PyTorch, Transformers, DistilBERT, NLP, SHAP

**Technologies:** Python, Apache Spark, XGBoost, Machine Learning, Data Analysis
